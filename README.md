# random-training

Web application built with Vue 3.

## Tech Stack

- **Frontend:** Vue 3 + TypeScript (Vite)
- **UI:** shadcn/ui for Vue + Tailwind CSS v4
- **Routing:** Vue Router
- **State:** Pinia
- **Code Quality:** ESLint + Prettier

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint & Format

```bash
npm run lint
npm run format
```
